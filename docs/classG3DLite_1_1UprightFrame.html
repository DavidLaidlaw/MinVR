Detailed Description

Coordinate frame expressed in Euler angles. Unlike a G3DLite::Quat, UprightFrame always keeps the reference frame from rolling about its own z axis. Particularly useful for cameras.

See Also
G3DLite::CoordinateFrame, G3DLite::Matrix4, G3DLite::PhysicsFrame, G3DLite::UprightSpline, G3DLite::UprightSplineManipulator
