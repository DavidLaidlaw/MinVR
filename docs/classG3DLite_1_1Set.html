Detailed Description

template<class T, class HashFunc = G3DLite::HashTrait<T>, class EqualsFunc = G3DLite::EqualsTrait<T>>
class G3DLite::Set< T, HashFunc, EqualsFunc >

An unordered data structure that has at most one of each element. Provides O(1) time insert, remove, and member test (contains).

Set uses G3DLite::Table internally, which means that the template type T must define a hashCode and operator== function. See G3DLite::Table for a discussion of these functions.
