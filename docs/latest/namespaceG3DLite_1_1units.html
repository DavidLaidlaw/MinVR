Detailed Description

Use using namespace G3DLite::units; to include all units into your program. The units system is specifically designed not to be general but to support commonly used units efficiently and clearly. See http://en.wikipedia.org/wiki/SI_prefix for interesting facts about SI/metric units and full definitions.
